# Быстрый старт

### 1. Клонирование и настройка

```bash
git clone https://github.com/heckfy811/hackhathon_max_bot.git
cd hackhathon_max_bot
cp .env.example .env
# Отредактируйте .env
```

### 2. Запуск

```bash
docker compose up -d
```

Через 10–15 секунд поднимется бд и накатятся миграции
